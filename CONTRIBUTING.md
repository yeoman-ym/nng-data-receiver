# 贡献指南

感谢您对本项目的关注！

## 如何贡献

1. Fork 本仓库
2. 创建您的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交您的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开一个 Pull Request

## 代码规范

- 遵循 C99 标准
- 保持代码简洁清晰
- 添加必要的注释
- 确保编译无警告

## 提交信息规范

- `feat`: 新功能
- `fix`: 修复 bug
- `docs`: 文档更新
- `style`: 代码格式调整
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具链相关

## 问题反馈

如遇到问题，请通过 GitHub Issues 反馈，并提供：
- 详细的问题描述
- 复现步骤
- 系统环境信息
- 相关日志输出
